fun <C : Collection<T>, T> C?.orEmptyList(): List<T>
Content copied to clipboard

This function takes a nullable C (which is a collection-based object) and returns an empty list, if it is actually null, or itself, if itself was not null. This function makes it easy to create new collections, if there is a nullable collection.

Return

An empty list, if C was null, or C itself, if it was not null
