skull

fun skull(    ownerName: String,     httpRequest: Boolean = true,     timeout: Duration = 5.seconds): Item
Content copied to clipboard

This function creates a new Item with a Material.PLAYER_HEAD. If the httpRequest parameter is set to true, the player's skin will be fetched from the Mojang API. If the httpRequest parameter is set to false, the player's skin will be fetched from the server.

Return

the created Item

Author

Fruxz

Since

1.0

See also

getMojangProfile

Parameters

ownerName

the name or uuid of the player

httpRequest

if true, the player's skin will be fetched from the Mojang API, otherwise from the server

timeout

time allowed, before the http request is aborted (useful if mojang is down again)
