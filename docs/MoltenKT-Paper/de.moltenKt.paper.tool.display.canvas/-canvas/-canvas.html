Parameters

key

The key of the canvas, that is used to bind the actions to the canvas. This should be unique for each canvas, but if you want to share a canvas habits & actions, use the same. Using shared keys can also improve memory-usage, if you do it cleverly. Why? because each registered action and session is identified by the key!

label

The label, which the viewer will see on top of the inventory.

canvasSize

The size of the canvas.

content

The content, which is placed inside the canvas

flags

The individual habits of the canvas.

openSoundEffect

The sound effect, which is played, when the canvas is opened. NOTE: If you want to edit a canvas, use the MutableCanvas class, maybe the toMutable function helps!
